# 自动记账 Skill

自动识别图片中的记账信息，并在一日记账APP中自动记录。

## 安装

```bash
cd ~/.openclaw/workspace/skills
git clone https://github.com/yaoyao-skills/auto-accounting.git
```

## 使用方法

发送包含账目信息的图片，摇摇会自动识别并记录：

- 微信/支付宝支付截图
- 京东、淘宝等购物平台订单
- 美团、饿了么外卖订单
- 消费小票、发票

## 依赖

- xiaoyi-image-understanding
- xiaoyi-gui-agent
