# 自动记账 Skill

> 自动识别图片中的记账信息，并在一日记账APP中自动记录

## 快速开始

### 安装

```bash
cd ~/.openclaw/workspace/skills
git clone https://github.com/taobaoaz/auto-accounting.git
```

### 使用

直接发送包含账目信息的图片即可：

- 微信/支付宝支付截图
- 京东、淘宝等购物平台订单
- 美团、饿了么外卖订单
- 消费小票、发票

## 支持平台

| 类型 | 平台 |
|------|------|
| 支付 | 微信支付、支付宝、银联 |
| 购物 | 京东、淘宝、拼多多、得物、唯品会 |
| 外卖 | 美团、饿了么 |
| 生鲜 | 盒马、叮咚买菜 |
| 其他 | 消费小票、银行转账 |

## 智能分类

- 餐饮：餐厅、外卖、咖啡、奶茶...
- 交通：打车、地铁、加油...
- 购物：淘宝、京东、超市...
- 娱乐：电影、游戏、会员...
- 医疗：医院、药店...
- 教育：课程、书籍...
